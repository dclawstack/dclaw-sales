# Getting Started with DClaw Sales

> Updated 2026-05-16 — v1.0.0-beta

Welcome to DClaw Sales — AI-powered sales pipeline management with lead tracking, opportunity Kanban, quotes, and pipeline analytics.

## What is DClaw Sales?

DClaw Sales helps sales teams:
- **Track leads** — Full lead lifecycle from new → contacted → qualified → lost
- **Manage pipeline** — Drag-and-drop Kanban board across 6 opportunity stages
- **Generate quotes** — Create, send, and track quotes tied to opportunities
- **Monitor performance** — Dashboard with pipeline value, win rate, and stage breakdown

## Quick Start (Local)

```bash
git clone https://github.com/dclawstack/dclaw-sales.git
cd dclaw-sales
docker compose up -d
open http://localhost:3017
```

## Prerequisites

- Docker (for local dev)
- DClaw Platform installed (for production)
- Kubernetes cluster access (for production)
